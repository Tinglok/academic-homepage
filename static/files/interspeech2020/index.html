Atss-Net: Target Speaker Separation via Attention-based Neural Network

This webpage is to show some listening examples for our Atss-Net method and the baseline VoiceFilter. It allows the network computing the correlation between each feature parallelly and automatically, and using shallower layers to extract more features, compared with pure CNNs and LSTMs methods. We also provide some real recording samples to prove Atss-Net can achieve good joint speech enhancement and separation performance.
